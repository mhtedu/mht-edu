# 棉花糖教育平台 - 开发进度文档

## 项目概述
基于 Taro + React + NestJS 的教育信息撮合平台，采用双向付费墙+分销裂变模式，连接家长、牛师、机构。

---

## 整体规划

### 核心业务流程
```
家长端：发布需求 → 牛师抢单 → 匹配成功 → 在线沟通 → 完成订单 → 评价
牛师端：查看需求 → 抢单 → 确认订单 → 授课 → 完成订单 → 收益
机构端：发布课程 → 管理牛师 → 招生 → 核销
```

## 开发规范

### 命名规范
- 页面文件：kebab-case（如 `publish-demand`）
- 组件：PascalCase（如 `DemandCard`）
- 变量/函数：camelCase（如 `handleSubmit`）
- 常量：UPPER_SNAKE_CASE（如 `API_BASE_URL`）

### Git提交规范
```
feat: 新功能
fix: 修复bug
style: 样式调整
refactor: 重构
docs: 文档更新
chore: 构建/工具链
```

### 代码规范
- 使用 Tailwind CSS，避免独立CSS文件
- 遵循 Airbnb 编码规范
- TypeScript 严格模式
- 所有API调用必须有console.log打印
