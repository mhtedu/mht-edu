# 快速部署指南（小白版）

## 🚀 5分钟快速部署

### 第一步：宝塔安装软件（2分钟）

登录宝塔面板，在【软件商店】搜索并安装：

1. **Nginx** - 直接点击安装
2. **MySQL 5.7** - 直接点击安装
3. **PM2管理器** - 直接点击安装
4. **Node版本管理器** - 安装后，再安装 Node.js 18

---

### 第二步：创建数据库（1分钟）

1. 点击【数据库】→【添加数据库】
2. 填写：
   - 数据库名：`mht_edu`
   - 用户名：`mht_edu`
   - 密码：自己设置（记住！）
3. 点击【提交】
4. 点击【管理】→ 进入phpMyAdmin
5. 点击【导入】，依次导入：
   - `server/database/init.sql`
   - `server/database/demo-data.sql`

---

### 第三步：上传项目（1分钟）

1. 在宝塔【文件】中，进入 `/www/wwwroot/`
2. 创建文件夹 `mht-edu`
3. 上传以下文件：
   - `server/` 文件夹（后端代码）
   - `dist-web/` 文件夹（前端打包）
   - `deploy/.env.production.example` 改名为 `.env.production`

---

### 第四步：配置并启动（1分钟）

**4.1 修改配置文件**

编辑 `/www/wwwroot/mht-edu/server/.env.production`：

```bash
DB_PASSWORD=你的数据库密码
JWT_SECRET=随机字符串32位以上
PROJECT_DOMAIN=https://你的域名.com
```

**4.2 安装后端依赖**

点击【终端】，执行：

```bash
cd /www/wwwroot/mht-edu/server
npm install --production
npm run build
```

**4.3 启动后端**

在【PM2管理器】中添加项目：
- 项目名称：`mht-edu-api`
- 启动文件：`/www/wwwroot/mht-edu/server/dist/main.js`
- 运行目录：`/www/wwwroot/mht-edu/server`

---

### 第五步：创建网站并配置Nginx

1. 点击【网站】→【添加站点】
   - 域名：你的域名
   - 根目录：`/www/wwwroot/mht-edu/dist-web`
   - PHP版本：纯静态

2. 点击网站【设置】→【配置文件】，粘贴以下内容：

```nginx
server {
    listen 80;
    server_name 你的域名.com;
    
    root /www/wwwroot/mht-edu/dist-web;
    index index.html;
    
    gzip on;
    gzip_types text/plain text/css application/javascript;
    
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        client_max_body_size 50m;
    }
}
```

3. 点击【保存】→【重载配置】

---

### 第六步：申请SSL证书

1. 点击网站【设置】→【SSL】→【Let's Encrypt】
2. 点击【申请】
3. 申请成功后开启【强制HTTPS】

---

## ✅ 验证部署

浏览器访问你的域名：
- ✅ 看到棉花糖教育首页 = 前端部署成功
- ✅ 访问 `你的域名/api/hello` 有响应 = 后端部署成功

## 🔧 常用命令

```bash
# 查看后端状态
pm2 status

# 查看后端日志
pm2 logs mht-edu-api

# 重启后端
pm2 restart mht-edu-api

# 重载Nginx
nginx -s reload
```

---

## 🆘 遇到问题？

### 网站打不开
1. 检查Nginx是否运行：`systemctl status nginx`
2. 检查端口：`netstat -tlnp | grep 80`

### API请求失败
1. 检查后端是否运行：`pm2 status`
2. 查看后端日志：`pm2 logs mht-edu-api`

### 数据库连接失败
1. 检查MySQL是否运行：`systemctl status mysqld`
2. 检查数据库密码是否正确
