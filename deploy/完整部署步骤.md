# 棉花糖教育平台 - 完整部署指南

## 📋 部署前准备

### 服务器环境要求
- ✅ 宝塔面板已安装
- ✅ Nginx 已安装（刚刚修复完成）
- ✅ MySQL 5.7/8.0 已安装
- ✅ Node.js 18+ 已安装
- ⚠️ PM2管理器（需要安装）

### 安装 Node.js 和 PM2（如果还没安装）

**在宝塔面板操作：**
1. 软件商店 → 运行环境 → Node版本管理器 → 安装

**或 SSH 安装：**
```bash

# 安装 PM2
npm install -g pm2

# 验证安装
node -v

```

## 步骤 2：创建数据库

### 方法一：宝塔面板操作（推荐）

1. **宝塔面板 → 数据库 → 添加数据库**
   - 数据库名：`mht_edu`
   - 用户名：`mht_edu`
   - 密码：自己设置（记住这个密码）
   - 字符集：`utf8mb4`

2. **导入数据库结构**
   - 点击 "管理" 进入 phpMyAdmin
   - 选择 `mht_edu` 数据库
   - 点击 "导入" → 选择 `/www/wwwroot/mht-edu/deploy/database/schema.sql`
   - 点击 "执行"

### 方法二：SSH 命令行

```bash
# 登录 MySQL
mysql -u root -p

# 创建数据库和用户
CREATE DATABASE mht_edu DEFAULT CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
CREATE USER 'mht_edu'@'localhost' IDENTIFIED BY '你的密码';
GRANT ALL PRIVILEGES ON mht_edu.* TO 'mht_edu'@'localhost';
FLUSH PRIVILEGES;
exit

# 导入数据库结构
mysql -u mht_edu -p mht_edu < /www/wwwroot/mht-edu/deploy/database/schema.sql
```

## 步骤 3：配置后端环境变量

```bash
# 创建环境配置文件
cat > /www/wwwroot/mht-edu/server/.env.production << 'EOF'
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=mht_edu
DB_USER=mht_edu
DB_PASSWORD=你的数据库密码

# 服务配置（使用 3002 端口，避免冲突）
PORT=3002
NODE_ENV=production

# JWT密钥（随机生成，至少32位）
JWT_SECRET=$(openssl rand -base64 32)

# 项目域名（改成你的域名）
PROJECT_DOMAIN=https://你的域名.com

# 微信小程序配置（先留空）
WECHAT_APPID=
WECHAT_SECRET=

# 对象存储配置（可选）
OSS_ACCESS_KEY=
OSS_SECRET_KEY=
OSS_BUCKET=
OSS_REGION=
OSS_ENDPOINT=
EOF

# 生成随机 JWT 密钥
sed -i "s/\$(openssl rand -base64 32)/$(openssl rand -base64 32)/" /www/wwwroot/mht-edu/server/.env.production
```

**查看配置：**
```bash
cat /www/wwwroot/mht-edu/server/.env.production
```

## 步骤 6：配置 Nginx 反向代理

### 方法一：宝塔面板配置（推荐）

1. **宝塔面板 → 网站 → 添加站点**
   - 域名：`你的域名.com`（如果有多个域名，用空格分隔）
   - 根目录：`/www/wwwroot/mht-edu/dist-web`
   - PHP版本：纯静态
   - 创建数据库：否（已创建）

2. **网站设置 → 反向代理 → 添加反向代理**
   - 代理名称：`mht-edu-api`
   - 目标URL：`http://127.0.0.1:3002`
   - 发送域名：`$host`

3. **修改配置文件（重要）**

   网站设置 → 配置文件，在 server {} 块内添加：

```nginx
# API 反向代理
location /api/ {
    proxy_pass http://127.0.0.1:3002;
    proxy_http_version 1.1;
    proxy_set_header Upgrade $http_upgrade;
    proxy_set_header Connection 'upgrade';
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
    proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    proxy_set_header X-Forwarded-Proto $scheme;
    proxy_cache_bypass $http_upgrade;
    
    # 文件上传大小限制
    client_max_body_size 50m;
}

# 前端静态文件
location / {
    root /www/wwwroot/mht-edu/dist-web;
    index index.html;
    try_files $uri $uri/ /index.html;
}

# 静态资源缓存
location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
    root /www/wwwroot/mht-edu/dist-web;
    expires 1y;
    add_header Cache-Control "public, immutable";
}
```

### 方法二：手动创建配置

```bash
# 创建 Nginx 配置文件
cat > /www/server/panel/vhost/nginx/你的域名.conf << 'EOF'
server {
    listen 80;
    server_name 你的域名.com www.你的域名.com;
    
    # 前端静态文件
    root /www/wwwroot/mht-edu/dist-web;
    index index.html;
    
    # API 反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:3002;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_cache_bypass $http_upgrade;
        client_max_body_size 50m;
    }
    
    # 前端路由
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
    
    access_log /www/wwwlogs/你的域名.log;
    error_log /www/wwwlogs/你的域名.error.log;
}
EOF

# 测试配置
nginx -t

# 重载配置
nginx -s reload
```

## 步骤 7：申请 SSL 证书

### 方法一：宝塔面板申请（推荐）

1. **网站设置 → SSL → Let's Encrypt**
2. 勾选域名
3. 点击 "申请"
4. 申请成功后，开启 "强制 HTTPS"

### 方法二：手动申请

```bash
# 安装 certbot
yum install -y certbot

# 申请证书
certbot certonly --webroot -w /www/wwwroot/mht-edu/dist-web -d 你的域名.com

# 证书路径
# /etc/letsencrypt/live/你的域名.com/fullchain.pem
# /etc/letsencrypt/live/你的域名.com/privkey.pem
```

## 🔧 常见问题

### 问题 1：后端启动失败

```bash
# 查看详细错误
pm2 logs mht-edu-api --lines 100

# 常见原因：
# 1. 数据库密码错误 → 检查 .env.production
# 2. 端口被占用 → netstat -tlnp | grep 3002
# 3. 依赖未安装 → npm install
```

### 问题 2：API 502 错误

```bash
# 检查后端是否运行
pm2 status

# 重启后端
pm2 restart mht-edu-api

# 查看 Nginx 错误日志
tail -50 /www/wwwlogs/你的域名.error.log
```

### 问题 3：前端页面空白

```bash
# 检查 dist-web 目录
ls -la /www/wwwroot/mht-edu/dist-web/

# 检查 Nginx 配置
cat /www/server/panel/vhost/nginx/你的域名.conf | grep root

# 检查浏览器控制台错误
```

### 问题 4：数据库连接失败

```bash
# 测试数据库连接
mysql -u mht_edu -p -h localhost mht_edu

# 检查 MySQL 状态
systemctl status mysqld

# 查看后端日志
pm2 logs mht-edu-api | grep -i mysql
```
