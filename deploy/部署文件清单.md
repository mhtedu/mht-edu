# 部署文件清单

## 📦 需要上传到服务器的文件

### 必须上传

```
/www/wwwroot/mht-edu/
├── server/                          # 后端代码（必须）
│   ├── dist/                        # 编译后的代码

│
├── dist-web/                        # 前端打包文件（必须）
│   ├── index.html
│   ├── js/
│   ├── css/
│   └── ...
│
└── deploy/                          # 部署脚本（可选）
    ├── 宝塔面板部署方案.md          # 详细部署文档
    ├── 快速部署指南.md              # 小白版指南
    ├── .env.production.example      # 环境配置模板
    ├── check-deploy.sh              # 部署检查脚本
    └── update.sh                    # 更新部署脚本
```

## 🚀 上传方式

### 方式一：宝塔面板上传（推荐小白）

1. 在本地将以下文件夹打包成zip：
   - `server/` → `server.zip`
   - `dist-web/` → `dist-web.zip`

2. 登录宝塔面板【文件】

3. 进入 `/www/wwwroot/mht-edu/`

4. 上传zip文件并解压

### 方式二：FTP/SFTP上传

使用FileZilla等工具，连接服务器后上传文件

### 方式三：Git拉取

```bash
cd /www/wwwroot/mht-edu
git clone 你的仓库地址 .
```

## 📝 配置文件示例

### .env.production

```bash
# 数据库配置（必填）
DB_HOST=localhost
DB_PORT=3306
DB_NAME=mht_edu
DB_USER=mht_edu
DB_PASSWORD=你的数据库密码

# JWT密钥（必填，随机32位以上字符串）
JWT_SECRET=abc123xyz789randomstringpleasechange

# 项目域名（必填）
PROJECT_DOMAIN=https://你的域名.com

# 服务配置
PORT=3000
NODE_ENV=production
```

## ⚡ 快速命令参考

```bash
# 后端安装依赖
cd /www/wwwroot/mht-edu/server
npm install --production

# 后端编译
npm run build

# 后端启动
pm2 start dist/main.js --name mht-edu-api

# 后端重启
pm2 restart mht-edu-api

# 查看日志
pm2 logs mht-edu-api

# 重载Nginx
nginx -s reload

# 检查部署状态
bash /www/wwwroot/mht-edu/deploy/check-deploy.sh
```

## 🆘 故障排查

| 问题 | 解决方案 |
|-----|---------|
| 网站打不开 | 检查Nginx状态：`systemctl status nginx` |
| API请求502 | 检查后端状态：`pm2 status` |
| 数据库连接失败 | 检查MySQL状态和密码 |
| 前端页面空白 | 检查dist-web目录是否完整 |
| 文件上传失败 | 修改Nginx `client_max_body_size` |
