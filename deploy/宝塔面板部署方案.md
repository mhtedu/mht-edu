## 准备工作

### 需要的信息
- 阿里云服务器IP地址
- 宝塔面板登录地址和账号密码
- 域名（已解析到服务器IP）

- 系统：CentOS 7.x 或 Ubuntu 20.04
- 内存：至少 2GB
- 硬盘：至少 20GB

## 宝塔面板配置

### 第一步：安装软件

登录宝塔面板后，在【软件商店】安装以下软件：

| 软件 | 版本 | 说明 |
|-----|------|------|
| Nginx | 1.24+ | Web服务器 |
| MySQL | 5.7 或 8.0 | 数据库 |
| PM2管理器 | 最新 | Node.js进程管理 |
| Node.js版本管理器 | 最新 | Node环境 |

### 第二步：安装Node.js

1. 点击【软件商店】→【Node.js版本管理器】→【设置】
2. 安装 Node.js 18.x 或 20.x LTS版本
3. 安装完成后，在终端执行：
```bash
node -v  # 查看Node版本
npm -v   # 查看npm版本
```

## 数据库配置

### 第一步：创建数据库

1. 点击【数据库】→【添加数据库】
2. 填写信息：
   - 数据库名：`mht_edu`
   - 用户名：`mht_edu`
   - 密码：自己设置一个复杂密码（记住这个密码！）
   - 字符集：`utf8mb4`

### 第二步：导入数据库

1. 点击数据库名称后的【管理】→ 打开phpMyAdmin
2. 选择 `mht_edu` 数据库
3. 点击【导入】→ 选择文件：
   - 先导入：`server/database/init.sql`（建表）
   - 再导入：`server/database/demo-data.sql`（演示数据）
   - 最后导入：`server/database/share-tables.sql`（分享功能表）

### 第三步：记录数据库信息

请记住以下信息，后面配置要用：
```
数据库地址：localhost（或 127.0.0.1）
数据库端口：3306
数据库名：mht_edu
数据库用户：mht_edu
数据库密码：你设置的密码
```

## 项目上传

### 第一步：创建网站目录

在宝塔面板【文件】中，进入 `/www/wwwroot/`，创建项目文件夹：
```
/www/wwwroot/mht-edu/
```

### 第二步：上传项目文件

将以下文件夹上传到服务器：

```
/www/wwwroot/mht-edu/
├── server/          # 后端代码（整个文件夹）
├── dist-web/        # 前端打包文件（整个文件夹）
└── .env.production  # 环境配置文件（需创建）
```

**上传方式（推荐）：**
1. 在宝塔面板【文件】中，进入 `/www/wwwroot/mht-edu/`
2. 将本地项目打包成 zip 文件
3. 上传 zip 文件，然后右键解压

## 后端部署

### 第一步：创建环境配置文件

在 `/www/wwwroot/mht-edu/server/` 目录下创建 `.env.production` 文件：

```bash
# 数据库配置
DB_HOST=localhost
DB_PORT=3306
DB_NAME=mht_edu
DB_USER=mht_edu
DB_PASSWORD=你的数据库密码

# 服务配置
PORT=3000
NODE_ENV=production

# JWT密钥（自己生成一个随机字符串）
JWT_SECRET=your-jwt-secret-key-change-this

# 微信小程序配置（如果有的话）
WECHAT_APPID=你的小程序AppID
WECHAT_SECRET=你的小程序Secret

# 项目域名（改成你的域名）
PROJECT_DOMAIN=https://你的域名.com
```

### 第二步：安装后端依赖

1. 点击【终端】
2. 进入后端目录并安装依赖：

```bash
cd /www/wwwroot/mht-edu/server
npm install --production
```

### 第三步：编译后端代码

```bash
cd /www/wwwroot/mht-edu/server
npm run build
```

### 第四步：使用PM2启动后端

在宝塔面板【软件商店】→【PM2管理器】→【设置】：

1. 点击【添加项目】
2. 填写信息：
   - 项目名称：`mht-edu-api`
   - 启动文件：`/www/wwwroot/mht-edu/server/dist/main.js`
   - 运行目录：`/www/wwwroot/mht-edu/server`
   - 端口：`3000`
3. 点击【提交】

或者在终端执行：
```bash
cd /www/wwwroot/mht-edu/server
pm2 start dist/main.js --name mht-edu-api
pm2 save
pm2 startup
```

验证后端是否启动成功：
```bash
curl http://localhost:3000/api/hello
```

## Nginx配置

### 第一步：创建网站

1. 点击【网站】→【添加站点】
2. 填写信息：
   - 域名：`你的域名.com`（如果有多个域名，用逗号分隔）
   - 根目录：`/www/wwwroot/mht-edu/dist-web`
   - PHP版本：纯静态（不需要PHP）
   - 创建数据库：否（已创建）

### 第二步：配置Nginx反向代理

点击网站名称→【设置】→【配置文件】，替换为以下内容：

```nginx
# 棉花糖教育平台 Nginx配置

server {
    listen 80;
    server_name 你的域名.com;  # 改成你的域名
    
    # 强制跳转HTTPS（SSL配置后取消注释）
    # return 301 https://$server_name$request_uri;
    
    # 前端静态文件
    root /www/wwwroot/mht-edu/dist-web;
    index index.html;
    
    # 开启gzip压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1k;
    gzip_comp_level 6;
    gzip_types text/plain text/css text/javascript application/javascript application/json application/xml;
    
    # 前端路由支持（Vue Router history模式）
    location / {
        try_files $uri $uri/ /index.html;
    }
    
    # API反向代理到后端
    location /api/ {
        proxy_pass http://127.0.0.1:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
        proxy_connect_timeout 60s;
        proxy_read_timeout 60s;
        proxy_send_timeout 60s;
        
        # 文件上传大小限制
        client_max_body_size 50m;
    }
    
    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
    
    # 禁止访问隐藏文件
    location ~ /\. {
        deny all;
    }
    
    # 访问日志
    access_log /www/wwwlogs/mht-edu.log;
    error_log /www/wwwlogs/mht-edu.error.log;
}

# HTTPS配置（SSL证书配置后取消注释）
# server {
#     listen 443 ssl http2;
#     server_name 你的域名.com;
#     
#     ssl_certificate /www/server/panel/vhost/cert/你的域名/fullchain.pem;
#     ssl_certificate_key /www/server/panel/vhost/cert/你的域名/privkey.pem;
#     ssl_protocols TLSv1.2 TLSv1.3;
#     ssl_ciphers HIGH:!aNULL:!MD5;
#     ssl_prefer_server_ciphers on;
#     ssl_session_cache shared:SSL:10m;
#     ssl_session_timeout 10m;
#     
#     # 其余配置同上...
# }
```

### 第三步：保存并重启Nginx

点击【保存】，然后点击【重载配置】

## SSL证书

### 方式一：宝塔免费证书（推荐）

1. 点击网站名称→【设置】→【SSL】
2. 选择【Let's Encrypt】
3. 勾选域名，点击【申请】
4. 申请成功后，开启【强制HTTPS】

### 方式二：阿里云免费证书

1. 登录阿里云控制台→【SSL证书】
2. 购买免费证书（每年20张）
3. 申请证书，填写域名信息
4. 下载Nginx格式证书
5. 在宝塔面板【SSL】→【其他证书】粘贴证书内容

## 常见问题

### 1. 数据库连接失败

**错误信息：** `ER_ACCESS_DENIED_ERROR: Access denied for user`

**解决方案：**
```bash
# 登录MySQL
mysql -u root -p

# 重新设置密码
GRANT ALL PRIVILEGES ON mht_edu.* TO 'mht_edu'@'localhost' IDENTIFIED BY '你的密码';
FLUSH PRIVILEGES;
```

### 2. 后端启动失败

**检查步骤：**
```bash
# 查看PM2日志
pm2 logs mht-edu-api

# 检查端口是否被占用
netstat -tlnp | grep 3000

# 手动启动测试
cd /www/wwwroot/mht-edu/server
node dist/main.js
```

### 3. 前端页面空白

**可能原因：**
- 打包文件未正确上传
- Nginx配置错误
- 路由配置问题

**解决方案：**
```bash
# 检查Nginx配置
nginx -t

# 查看Nginx错误日志
tail -f /www/wwwlogs/mht-edu.error.log
```

### 4. API请求502错误

**原因：** 后端服务未启动或端口不对

**解决方案：**
```bash
# 检查后端是否运行
pm2 status

# 检查端口
curl http://localhost:3000/api/hello

# 重启后端
pm2 restart mht-edu-api
```

### 5. 文件上传失败

**解决方案：**

修改Nginx配置：
```nginx
client_max_body_size 50m;  # 增加上传大小限制
```

修改PHP配置（如果用到）：
```ini
upload_max_filesize = 50M
post_max_size = 50M
```

### 6. 微信小程序请求失败

**原因：** 需要在微信小程序后台配置服务器域名

**解决方案：**
1. 登录微信小程序后台
2. 【开发】→【开发管理】→【开发设置】→【服务器域名】
3. 添加 `https://你的域名.com`
