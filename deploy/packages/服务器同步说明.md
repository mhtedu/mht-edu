## 服务器目录结构
```
/www/wwwroot/mht-edu/
├── dist/                  # 前端 H5 文件 (站点目录)
│   ├── index.html         # 入口文件
│   ├── js/                # JavaScript 文件
│   ├── css/               # 样式文件
│   └── admin/             # 管理后台
│       ├── admin.html
│       ├── admin.css
│       ├── admin.js
│       └── login.html
├── dist-weapp/            # 小程序文件
└── server/                # 后端服务 (执行目录)
    ├── dist/              # 编译后的 JS
    └── src/               # 源代码
```

## 部署步骤

### 方法1：宝塔面板上传

1. 登录宝塔面板
2. 进入 文件 → `/www/wwwroot/mht-edu/`
3. 上传 `mht-edu-sync-20260404_082622.tar.gz`
4. 在宝塔终端执行：
   ```bash
   cd /www/wwwroot/mht-edu
   tar -xzvf mht-edu-sync-20260404_082622.tar.gz
   # 解压后文件会覆盖到正确目录
   ```
5. 重启后端服务：
   ```bash
   cd /www/wwwroot/mht-edu/server
   pm2 restart mht-edu-server
   ```

### 方法2：SCP 命令上传 (如有 SSH 访问)

```bash
# 本地执行
scp deploy/packages/mht-edu-sync-20260404_082622.tar.gz root@119.91.193.179:/www/wwwroot/mht-edu/

# 服务器执行
ssh root@119.91.193.179
cd /www/wwwroot/mht-edu
tar -xzvf mht-edu-sync-20260404_082622.tar.gz
pm2 restart mht-edu-server
```
