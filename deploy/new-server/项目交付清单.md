## 🚀 快速部署

### 步骤1: 上传部署包

```bash
# 方式1: SCP上传
scp deploy/packages/mht-edu-deploy-*.tar.gz root@119.91.193.179:/www/wwwroot/

# 方式2: 宝塔面板上传
# 登录宝塔面板 → 文件管理 → 上传文件
```

### 步骤2: 解压

```bash
cd /www/wwwroot
tar -xzf mht-edu-deploy-*.tar.gz
mv mht-edu-deploy mht-edu
```

### 步骤3: 配置环境

```bash
cd /www/wwwroot/mht-edu
cp .env.example server/.env
nano server/.env

# 必填项:
# - DB_PASSWORD (数据库密码)
# - WECHAT_APPID (小程序AppID)
# - WECHAT_SECRET (小程序Secret)
```

### 步骤4: 运行部署脚本

```bash
chmod +x deploy.sh
./deploy.sh
```
