## 五、前端部署

### 5.1 配置Nginx

```bash
# 宝塔面板 - 网站 - wx.dajiaopei.com - 设置 - 配置文件

# 替换为以下配置:
server {
    listen 80;
    server_name wx.dajiaopei.com;
    root /www/wwwroot/mht-edu/dist-web;
    index index.html;

    # 开启gzip压缩
    gzip on;
    gzip_vary on;
    gzip_min_length 1024;
    gzip_types text/plain text/css text/xml text/javascript application/javascript application/json application/xml;

    # H5前端
    location / {
        try_files $uri $uri/ /index.html;
    }

    # API反向代理
    location /api/ {
        proxy_pass http://127.0.0.1:3002;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_cache_bypass $http_upgrade;
    }

    # PC管理后台
    location /admin/ {
        alias /www/wwwroot/mht-edu/admin/;
        try_files $uri $uri/ /admin/admin.html;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2|ttf|eot)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }

    # 禁止访问敏感文件
    location ~ /\. {
        deny all;
    }
    location ~ /node_modules/ {
        deny all;
    }
}

# 保存后重载Nginx
nginx -t
nginx -s reload
```

### 5.2 SSL证书配置

```bash
# 方式1: 宝塔面板 - 网站 - SSL - Let's Encrypt - 申请证书

# 方式2: 手动配置
server {
    listen 443 ssl http2;
    server_name wx.dajiaopei.com;
    
    ssl_certificate /www/server/panel/vhost/cert/wx.dajiaopei.com/fullchain.pem;
    ssl_certificate_key /www/server/panel/vhost/cert/wx.dajiaopei.com/privkey.pem;
    ssl_protocols TLSv1.2 TLSv1.3;
    ssl_ciphers HIGH:!aNULL:!MD5;
    
    # 其余配置同上...
}

# HTTP重定向到HTTPS
server {
    listen 80;
    server_name wx.dajiaopei.com;
    return 301 https://$server_name$request_uri;
}
```

## 八、常见问题

### 8.1 端口占用

```bash
# 查看3002端口占用
lsof -i:3002

# 杀掉进程
kill -9 <PID>

# 重启服务
pm2 restart mht-edu-server
```

### 8.2 数据库连接失败

```bash
# 检查MySQL状态
systemctl status mysql

# 检查用户权限
mysql -u root -p
SELECT User, Host FROM mysql.user WHERE User='mht_edu';

# 如果没有localhost权限，重新授权
GRANT ALL PRIVILEGES ON mht_edu.* TO 'mht_edu'@'localhost';
FLUSH PRIVILEGES;
```

### 8.3 Nginx 502错误

```bash
# 检查后端服务是否运行
pm2 list

# 检查Nginx配置
nginx -t

# 查看错误日志
tail -f /www/wwwlogs/wx.dajiaopei.com.error.log
```

### 8.4 API跨域问题

```bash
# 已在后端配置CORS，如仍有问题，在Nginx添加:
location /api/ {
    add_header 'Access-Control-Allow-Origin' '*';
    add_header 'Access-Control-Allow-Methods' 'GET, POST, PUT, DELETE, OPTIONS';
    add_header 'Access-Control-Allow-Headers' 'DNT,X-Mx-ReqToken,Keep-Alive,User-Agent,X-Requested-With,If-Modified-Since,Cache-Control,Content-Type,Authorization';
    
    if ($request_method = 'OPTIONS') {
        return 204;
    }
    
    proxy_pass http://127.0.0.1:3002;
}
```
