# 棉花糖教育平台 - 数据库导入指南

## 文件位置

```
/www/wwwroot/mht-edu/server/database/
├── mht_edu_all_in_one.sql  # 完整数据库脚本（一键导入）
└── import_db.sh            # 自动导入脚本
```

---

## 快速导入（推荐）

### 方式一：一行命令

```bash
mysql -u root -pmht2026edu < /www/wwwroot/mht-edu/server/database/mht_edu_all_in_one.sql
```

### 方式二：使用导入脚本

```bash
cd /www/wwwroot/mht-edu/server/database
chmod +x import_db.sh
./import_db.sh
```

### 方式三：phpMyAdmin 导入

1. 宝塔面板 → 数据库 → 管理 → 进入 phpMyAdmin
2. 点击左侧选择或创建数据库 `mht_edu`
3. 顶部【导入】→ 选择文件 `mht_edu_all_in_one.sql`
4. 点击【执行】

## 导入后配置

修改后端环境变量：

```bash
vi /www/wwwroot/mht-edu/server/.env
```

确保数据库配置正确：

```env
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=mht2026edu
DB_DATABASE=mht_edu
```

重启服务：

```bash
cd /www/wwwroot/mht-edu/server
pm2 restart mht-edu-server
```

## 常见问题

### Q1: 提示 "Unknown database 'mht_edu'"
SQL文件已包含创建数据库语句，直接导入即可。

### Q2: 提示 "Access denied"
确认密码是否为 `mht2026edu`，或在宝塔面板查看正确的 root 密码。

### Q3: 如何完全重置数据库？
重新导入 SQL 文件即可（已包含删除所有表的语句）。
