## 三、抢单/匹配通知流程 ✅ 新增

### 3.1 抢单通知流程
```
1. 老师抢单成功
   ↓
2. 创建抢单记录(order_matches)
   ↓
3. 发送站内提醒(messageService.createReminder)
   ↓
4. 发送微信订阅消息给家长
   ↓
5. 发送短信通知给家长
```

### 3.2 匹配成功通知流程
```
1. 家长选择老师
   ↓
2. 更新订单状态为"已匹配"
   ↓
3. 发送站内消息给老师
   ↓
4. 发送微信订阅消息给老师(含家长联系方式)
   ↓
5. 发送短信通知给老师
```

### 3.3 订单状态变更通知
```
状态变更 → 发送站内消息 → 发送微信订阅消息 → 发送短信
```

## 八、待配置项

### 8.1 微信订阅消息模板ID
需要在微信公众平台申请以下模板：
```
TEMPLATE_ID_ORDER_GRABBED    # 抢单成功通知
TEMPLATE_ID_ORDER_MATCHED    # 匹配成功通知
TEMPLATE_ID_ORDER_STATUS     # 订单状态变更
TEMPLATE_ID_ORDER_CANCELLED  # 订单取消通知
```

### 8.2 短信服务配置
在 site_config 表中配置：
```
sms_access_key_id      # 阿里云AccessKey
sms_access_key_secret  # 阿里云Secret
sms_sign_name          # 短信签名
sms_template_code      # 短信模板ID
sms_enabled            # 启用状态(1启用)
```

### 8.3 微信小程序配置
在 site_config 表中配置：
```
wx_appid      # 小程序AppID
wx_secret     # 小程序Secret
```
