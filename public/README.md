# 独立PC端管理后台使用说明

## 📁 文件说明

这是一个**完全独立的PC端管理后台**，不依赖Taro框架，专为PC端浏览器设计。

### 文件列表
```
public/
├── admin.html    # 管理后台主页面
├── admin.css     # 样式文件
└── admin.js      # 功能脚本
```

---

## 🚀 部署步骤

### 方式一：直接访问（推荐）

这三个文件已经包含在H5前端编译中，访问地址：

```
https://mt.dajiaopei.com/admin.html
```

**如果无法访问，请检查：**

1. 文件是否上传到 `/www/wwwroot/mht-edu/dist/` 目录
2. Nginx配置是否正确

---

### 方式二：独立部署

如果希望管理后台独立部署，可以：

1. 在服务器创建独立目录：
```bash
mkdir -p /www/wwwroot/admin.mt.dajiaopei.com
```

2. 上传三个文件到该目录

3. 配置独立的Nginx站点（可选）

## 🔧 配置API地址

管理后台默认连接到 `/api/admin`，如果您的后端API地址不同，请修改 `admin.js` 文件：

```javascript
const CONFIG = {
    API_BASE: '/api/admin', // 修改为您的API地址
    PAGE_SIZE: 20,
};
```

## 🔐 安全建议

当前版本**没有登录验证**，建议：

1. **添加管理员登录**
   - 创建管理员表
   - 实现JWT验证
   - 添加权限管理

2. **限制访问IP**
   - 在Nginx中配置IP白名单
   - 只允许特定IP访问

3. **HTTPS强制**
   - 确保SSL证书有效
   - 强制HTTPS访问
