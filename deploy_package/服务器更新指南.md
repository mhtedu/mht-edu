# 服务器更新指南

## 第一步：下载文件

从以下位置下载文件：
- `/workspace/projects/deploy_package/mht-edu-sync-20260404_082622.tar.gz` (2.5M)
- `/workspace/projects/deploy_package/server-update.sh` (3KB)

## 第二步：上传到服务器

### 方法1：宝塔面板上传

1. 登录宝塔面板
2. 文件 → 进入 `/www/wwwroot/mht-edu/`
3. 上传两个文件

### 方法2：命令行上传

```bash
scp mht-edu-sync-20260404_082622.tar.gz root@119.91.193.179:/www/wwwroot/mht-edu/
scp server-update.sh root@119.91.193.179:/www/wwwroot/mht-edu/
```

## 第三步：执行更新

### 方法1：执行脚本（推荐）

```bash
cd /www/wwwroot/mht-edu
chmod +x server-update.sh
./server-update.sh
```

### 方法2：手动解压

```bash
cd /www/wwwroot/mht-edu

# 解压同步包
tar -xzvf mht-edu-sync-20260404_082622.tar.gz

# 移动文件到正确位置
# 前端文件 → dist/
cp -r mht-edu-sync/dist/* dist/

# 管理后台 → dist/admin/
mkdir -p dist/admin
cp -r mht-edu-sync/dist/admin/* dist/admin/

# 小程序 → dist-weapp/
mkdir -p dist-weapp
cp -r mht-edu-sync/dist-weapp/* dist-weapp/

# 后端 → server/
cp -r mht-edu-sync/server/dist/* server/dist/
cp -r mht-edu-sync/server/src/* server/src/

# 清理临时文件
rm -rf mht-edu-sync

# 重启后端服务
pm2 restart mht-edu-server
```

## 第四步：验证更新

```bash
# 检查前端
curl -I https://wx.dajiaopei.com/

# 检查 API
curl https://wx.dajiaopei.com/api/hello
```
